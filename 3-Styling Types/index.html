Styling Types

There are three ways to styles our web pages

  1. Inline Styling
  2. Internal Styling
  3. External Styling

Inline Styling

This line is styled using inline styling--(using style attribute on element)

Use style attribute on element for inline styling

Internal Styling

This line is styled using internal styling --(using style tag in head of html file

Use style tag in head part of html file for internal styling

External Styling

This part is styled using styles from external styles.css file

Use Link tag in head part of html file to link styles files with html file
